# nbc-task-job-change-system
### [내일배움캠프 Unreal 7기] 과제2. 전직 시스템과 전투 시스템

## 📌 프로젝트 개요 (Project Overview)
*   **주제**: 객체지향 프로그래밍(OOP)을 활용한 직업 선택 및 전투 로직 구현
*   **개발 환경**: C++ (Visual Studio)
*   **핵심 기술**:
    *   추상 클래스 및 순수 가상 함수 (Pure Virtual Function)
    *   클래스 상속 및 오버라이딩 (Overriding)
    *   동적 할당을 활용한 다형성 구현

## 🛠 기능 명세 (Features)

### 1. 플레이어 직업 전직 시스템
`Player`라는 기본 추상 클래스를 상속받아, 4가지의 직업 클래스를 구현.
*   **Player (Base)**: `attack()` 순수 가상 함수를 포함하는 추상 클래스.

### 2. 몬스터
모든 몬스터는 고정된 능력치를 가진다.
*   **기본 능력치**: HP 10, 공격력 30, 방어력 10, 스피드 10
*   **공격 로직**: 플레이어를 공격하여 HP를 감소시키는 로직 구현.

### 3. 전투 로직 (Combat Logic)
플레이어와 몬스터는 턴제 방식으로 전투를 진행.

*   **데미지 계산 공식**: `(자신의 공격력 - 적의 방어력)`
*   **최소 데미지 보정**: 계산된 데미지가 0 이하일 경우, 최소 데미지를 **1**로 설정.
*   **직업별 공격 방식**:
    *   **전사 / 마법사**: 계산된 데미지로 **1회** 공격.
    *   **궁수**: `(데미지 / 3)`의 데미지로 **3회** 공격. (단, 몫이 0이면 1로 보정)
    *   **도적**: `(데미지 / 5)`의 데미지로 **5회** 공격. (단, 몫이 0이면 1로 보정)
